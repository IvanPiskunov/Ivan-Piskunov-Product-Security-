# Ivan Piskunov - Dual-Track Site v2

Compact GitHub Pages build for a dual-track professional profile:
- Product Security leadership
- Practice-led education / Professor of Practice-aligned positioning

## Files
- `index.html` - main landing page
- `press-kit.html` - media/speaking/academic partnership page
- `styles.css` - site styles
- `script.js` - small navigation helper
- `assets/headshot-placeholder.svg` - replace with your official photo
- `downloads/` - bios, speaking topics, media notes

## Deploy on GitHub Pages
1. Create a new repository.
2. Upload all files from this folder.
3. In GitHub: Settings -> Pages.
4. Choose `Deploy from a branch`.
5. Select the `main` branch and `/root`.
6. Save. GitHub Pages will publish the site.
